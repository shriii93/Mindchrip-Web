# MindChirps - Blog & Article Hub

A modern, responsive blog platform built with HTML, CSS, and JavaScript featuring articles about books, movies, technology, psychology, and lifestyle topics.

## 🌟 Features

- **Responsive Design** - Mobile-friendly layout that adapts to all screen sizes
- **Dynamic Navigation** - Sticky header with dropdown category menu
- **Article Cards** - Beautiful grid-based card layout for articles with images and descriptions
- **Search Functionality** - Built-in search bar in the header
- **User Authentication** - Modal-based login system
- **Newsletter Subscription** - Email subscription feature
- **Pagination** - Navigate through multiple pages of articles
- **Smooth Animations** - Floating card animations and smooth scrolling
- **Modern UI** - Gradient backgrounds, shadows, and hover effects

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No installation required - purely front-end

### Installation

1. **Clone or download** the project files
2. **Save the files** in the same folder:
   - `index.html`
   - `Style.css`
   - `Script.js`
3. **Open** `index.html` in your web browser

## 💻 Usage

### Login Modal
Click the "Login" button in the header to open the login modal. Close it by:
- Clicking the X button
- Clicking outside the modal

### Pagination
Click on page numbers (1-4) or "Next" button to switch between article pages. The active page is highlighted in red.

### Newsletter Subscription
Enter your email and click "Subscribe" to receive a confirmation message.

### Search
Use the search bar to search for articles (currently logs to console).

### Hero Button
Click the "Explore" button to smoothly scroll down to the articles section.
